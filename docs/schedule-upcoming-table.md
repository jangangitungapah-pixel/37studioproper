# Schedule Upcoming Table

Halaman Schedule sekarang memiliki tabel:

```txt
Jadwal Mendatang
```

Tujuan:

```txt
Menampilkan semua booking mendatang, termasuk paket tanpa durasi yang tidak muncul sebagai blok calendar grid.
```

Behavior:

```txt
1. Booking cancelled / void / deleted tidak ditampilkan.
2. Client request rejected / cancelled tidak ditampilkan.
3. Paket tanpa durasi tetap muncul di tabel dengan badge Tanpa blok kalender.
4. Klik customer / kode booking membuka detail booking.
5. Calendar grid tetap fokus ke booking yang memakai slot studio utama.
```
