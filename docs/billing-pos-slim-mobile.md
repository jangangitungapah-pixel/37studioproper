# Billing POS Slim Mobile UI

Halaman Billing / POS dirampingkan untuk mobile.

Perubahan:

```txt
1. Summary card atas dibuat lebih pendek.
2. Kas masuk dibuat lebih compact.
3. Search dan filter dibuat lebih tipis.
4. Reminder tagihan dibuat seperti compact follow-up list.
5. Billing row tidak lagi memakai 3 kotak nominal besar.
6. Total, Dibayar, dan Sisa menjadi baris nominal tipis.
7. Tombol Reminder, Invoice, Bayar, Void dibuat lebih pendek.
8. Font bold dikurangi agar visual lebih ringan.
```

Scope:

```txt
CSS-only.
Tidak mengubah data, invoice, payment logic, void logic, reminder link, atau Firestore.
```
