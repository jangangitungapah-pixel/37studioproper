# Booking Detail Modal Slim

Modal detail booking dirampingkan untuk mobile.

Perubahan:

```txt
1. Header lebih pendek.
2. Judul booking single-line ellipsis.
3. Informasi booking menjadi quick facts.
4. Ringkasan harga hanya menampilkan Total, Terbayar, dan Sisa.
5. Paket tanpa durasi tampil sebagai "tanpa blok kalender".
6. Metadata tetap collapsed.
7. Footer action lebih tipis.
```

Tujuan:

```txt
Modal detail terasa seperti quick inspection sheet, bukan invoice panjang.
```

## BDM-1B Readable Slim

Perbaikan lanjutan modal detail booking:

```txt
1. Mengurangi font bold berlebihan.
2. Judul booking tidak dipaksa satu baris.
3. Value panjang seperti layanan dan waktu boleh wrap.
4. Grid informasi booking di mobile berubah jadi list ramping.
5. Ringkasan harga tetap jelas tanpa typography terlalu berat.
```

## BDM-1C Mobile Full Width

Perbaikan lanjutan untuk modal detail booking mobile:

```txt
1. Panel mobile dibuat 100vw dan 100dvh.
2. Border radius panel mobile dihilangkan agar full kanan-kiri.
3. Font bold dikurangi lagi.
4. Informasi booking dibuat lebih tipis.
5. Value panjang boleh wrap agar tidak kepotong.
6. Footer action lebih pendek.
```
