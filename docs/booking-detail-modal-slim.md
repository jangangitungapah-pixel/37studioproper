# Booking Detail Modal Slim

Modal detail booking dirampingkan untuk mobile.

Perubahan:

```txt
1. Header lebih pendek.
2. Judul booking single-line ellipsis.
3. Informasi booking menjadi quick facts.
4. Ringkasan harga hanya menampilkan Total, Terbayar, dan Sisa.
5. Paket tanpa durasi tampil sebagai "tanpa blok kalender".
6. Metadata tetap collapsed.
7. Footer action lebih tipis.
```

Tujuan:

```txt
Modal detail terasa seperti quick inspection sheet, bukan invoice panjang.
```
